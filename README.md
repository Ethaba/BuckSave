# BuckSave

BuckSave is a personal budget tracker Android application developed in Kotlin using XML layouts. The app helps users manage spending by setting monthly budget goals, creating custom categories, logging expenses, viewing reports, and earning rewards.

## Features

- User registration and login
- Local SQLite database storage
- Set minimum and maximum monthly spending goals
- Create custom expense categories
- Add expense entries with:
  - amount
  - date
  - start time
  - end time
  - category
  - description
  - optional receipt image
- View expense list
- View receipt images
- Generate spending reports by selected period
- View category spending breakdown
- Dashboard with budget progress
- Category budget usage tracking
- Rewards and achievement system
- User profile screen

## Technologies Used

- Kotlin
- XML layouts
- Android Studio
- SQLiteOpenHelper
- GitHub

## How to Run the App

1. Clone this repository.
2. Open the project in Android Studio.
3. Allow Gradle to sync.
4. Connect an Android device or start an emulator.
5. Click Run.
