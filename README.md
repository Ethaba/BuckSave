# BuckSave

BuckSave is a smart personal budgeting Android application developed using Kotlin and XML in Android Studio. The app helps users manage their finances by tracking expenses, setting spending goals, monitoring category budgets, generating visual reports, and earning rewards through a gamified budgeting system.

---

# App Overview

BuckSave was designed to help users develop better financial habits through budgeting tools, smart spending analysis, and interactive financial tracking features.

The application allows users to:
- Set minimum and maximum monthly spending goals
- Create category budgets
- Track expenses
- Monitor spending behaviour
- Generate visual spending reports
- Receive budgeting insights
- Earn rewards and achievements for responsible budgeting

# Advanced Features

## Smart Budget Insights
The application includes a dedicated Insights screen that provides:
- Budget Health Score
- Smart financial feedback
- Overspending warnings
- Category health monitoring
- Spending behaviour analysis

This feature helps users better understand their financial habits and identify risky spending behaviour.

## Gamified Rewards System
BuckSave includes a gamification system that rewards responsible budgeting behaviour.

Users can:
- Earn points
- Unlock achievement badges
- Progress through financial levels
- Track reward progress
- View earned and locked achievements

Levels include:
- Beginner
- Rookie
- Budget Builder
- Saver
- Budget Master

## Dynamic Budget Progress Tracking
The dashboard progress bar changes color based on the user's spending condition:
- Orange → Below minimum goal
- Blue/Green → Healthy spending
- Red → Overspending

## Interactive Graph Visualization
The Reports screen includes a dynamic Pie Chart that:
- Displays spending per category
- Updates according to selected date range
- Visualizes financial distribution
- Displays spending relative to budgeting goals
